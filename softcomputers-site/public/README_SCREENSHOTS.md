## Recommended Dimensions
- **iPhone 14 Pro**: 1170 x 2532 pixels
- **iPhone 15 Pro**: 1179 x 2556 pixels
- **Aspect Ratio**: Approximately 9:19.5

## How to Take Screenshots
1. Open FocusFlow app in iOS Simulator or on your iPhone
2. Navigate to each screen
3. Take screenshot (Cmd+S in Simulator, or Volume Up + Power on iPhone)
4. Save with the exact filenames listed above
5. Place all files in `/public/images/` directory

## Notes
- The iPhone simulator component will automatically use these images when available
- If a screenshot is missing, it will show a beautiful gradient fallback
- Users can navigate between screenshots using the dots below the phone
